## installation

To get a local copy up and running follow these simple example steps.

- clone the repository by running
``` https://github.com/AckonSamuel/bookstore.git ```
- navigate to the folder
``` cd <DIRECTORY> ```
- Install packages
``` npm i ```
- Run the app
``` npm start ```
